# Unstrung, An Alternate Control Game

_Unstrung_ is a game where one must cast shadows on the environment in order to 
traverse hazardous levels filled with puzzles. The manipulation of shadows within
the real world combines the physical interaction of a projector and camera(s) with
digital feedback, allowing for unique player movement, level exploration, and event
triggers that encourage innovative problem solving.

This game was designed for the as part of the one term major qualifying project at Worcester Polytechnic Institute
in partial fulfillment of the requirements for the Degrees of Bachelor of Science and Bachelor of Arts in Interactive
Media and Game Design and Computer Science.

## Github
Available Branches include main, the current working branch, final-test, including level decorations, and Releases, storing the most recent operational build.

Attached to each release includes a compiled build in zip format with all required files/folder hierarchy.

## Equipment Setup
To play _Unstrung_ in it's intended format with shadow detection, a 1080p camera and projector are required. The projector must be set up pointed straight at a wall
with the camera capturing the entire display. The camera is recommend to be placed two to three feet away from the wall, vertically below the projector such that a large 
portion of the projection is uninterrupted by the camera's field of view.

## Operation
Begin _Unstrung_ by first running the compiled game .exe within its provided folder. Ensure the game displays on the projector. Next, begin the python script ShadowSocket.py
within the folder "External Software". When the checkboard pattern displays on the projector focus the second window that appears displaying the camera feed. Press spacebar to
capture the detected corners and Escape to exit camera calibration mode. Refocus _Unstrung_ and begin playing!

## Controls
_Unstrung_ can be played with both keyboard and controller:
- X or Left bumper -> Lock/Unlock actors (activating shadow detection)
- C or Left Trigger -> Confirm Shadow Placement
- Backspace or Left D-pad -> Remove Shadows

## Other Ways to Play
_Unstrung_ can also be played in a debug mode. Keyboard controls remain the same, however the mouse can be used to place vertices to draw shadows. In addition, checkpoints can be
skipped to using the number keys.
